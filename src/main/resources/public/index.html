Canvas Calendar Import

This is a simple tool that allows Teachers/Course Designers/TAs to import a CSV spreadsheet of events into the calendar for a Canvas course, this batch of events can then be deleted in bulk at some point later.

Configuration

The configuration for this tool is in the documentation and uses a LTI 1.3 launch service to handle the launch from Canvas.

Support
